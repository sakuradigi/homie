# Changelog

版本號格式：`v主版本.功能迭代.小修正`

---

## v0.4.0 — 2026-07-02

### 新增
- 圖片上傳前自動壓縮（長邊 1600px、JPEG 0.85）：費用大幅降低、上傳更快
- API 錯誤訊息友善化（429 顯示「請求太頻繁」）
- DOMPurify 淨化 AI 回應後再渲染（防 XSS）

### 修正
- 點擊歷史紀錄恢復時不再重複寫入歷史（原本每點一次多存一筆）
- 朗讀語速／語音選單改為單一全域控制（原本僅第一頁的選單有效，且產生重複 id）
- 英文繪本歷史紀錄恢復後維持英文朗讀（原本一律變中文）
- 繪本分頁辨識容忍空格與英文變體（`=== 第 1 頁 ===`、`=== Page 1 ===`）
- 語音清單改用 `voiceschanged` 事件載入（原本賭 300ms 時序）
- 空白 API Key 不再顯示「已儲存」
- 切換分頁時自動停止朗讀
- Gemini Key 改走 header 不再出現在 URL；max_tokens 提高至 8192 避免多頁截斷

### 移除
- 半套的英文介面切換（多處文案未翻譯，維護成本大於價值）
- 死碼：`escapeHtml()`、`updateSpeed()`、`buildTTSControls()`
